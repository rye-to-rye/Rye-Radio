## How to Use ğŸš€

1. **Search for Stations**: 
   - Type the name of a radio station in the search box
   - Press `Enter` to search

2. **Navigate Results**:
   - Use the **Previous** and **Next** buttons to browse through search results
   - Buttons become disabled when you reach the beginning or end of results

3. **Listen**:
   - The station starts playing automatically
   - Station name is displayed above the audio player
   - Use the native audio controls for volume and playback
